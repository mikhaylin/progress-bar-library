Progress Bar Library

A pure CSS library for creating progress bars.

Basic progress bar (refactored)

Simplified HTML using pseudo-element:

<div class="progress-bar" style="--pb-progress: 50%;"></div>

Improvement: HTML simplified from 2 elements to 1

With CSS Custom Properties

Example using CSS variables for customization:

<div class="progress-bar" style="
    --pb-color: #e74c3c;
    --pb-background: #ffebee;
    --pb-progress: 75%;
"></div>

Available CSS variables:

  • --pb-color - fill color (default: #3498db)
  • --pb-background - background color (default: #f0f0f0)
  • --pb-progress - progress value (default: 0%)

Themes: Blue Theme

First theme using CSS class:

<div class="progress-bar theme-blue" style="--pb-progress: 60%;"></div>

Features:

  • Custom blue color palette
  • Gradient effect on fill
  • Light blue background

Themes: Green Theme

Green theme with shadow effect:

<div class="progress-bar theme-green" style="--pb-progress: 80%;"></div>

Features:

  • Green color palette
  • Gradient with darker shade
  • Inner shadow for depth effect

Themes: Dark Theme

Dark theme with cyan accent for better visibility:

<div class="progress-bar theme-dark" style="--pb-progress: 70%;"></div>

Features:

  • Cyan color on dark gray background (high contrast)
  • Glow effect around fill
  • Inset shadow for depth
  • Border for definition

Animated Transitions

Smooth animation when progress changes (0.3s duration):

/* CSS transition */
.progress-bar::after {
    transition: width 0.3s ease;
}

/* JavaScript to change progress */
element.style.setProperty('--pb-progress', '75%');

Features:

  • Smooth 0.3s animation on width change
  • Ease timing function for natural motion
  • Works with all themes and modifiers
  • No JavaScript required for animation

Note: Animation works when CSS variable --pb-progress changes via JavaScript or developer tools.

JavaScript Integration

Optional JavaScript helper for easier control:

// Basic usage
ProgressBar.setProgress('#my-bar', 75);

// Animated transition
ProgressBar.animateTo('#my-bar', 100, 1500);

// Get current value
const current = ProgressBar.getProgress('#my-bar');

// Simulate loading (demo)
ProgressBar.simulateLoading('#my-bar');

API Methods:

  • setProgress(selector, value) - Set progress immediately
  • animateTo(selector, value, duration) - Smooth animation
  • getProgress(selector) - Get current value
  • simulateLoading(selector, interval) - Demo simulation

Note: JavaScript is optional. The library works with pure CSS.
