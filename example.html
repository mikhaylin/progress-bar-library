Progress Bar Library

A pure CSS library for creating progress bars.

Basic progress bar (refactored)

Simplified HTML using pseudo-element:

<div class="progress-bar" style="--pb-progress: 50%;"></div>

Improvement: HTML simplified from 2 elements to 1

With CSS Custom Properties

Example using CSS variables for customization:

<div class="progress-bar" style="
    --pb-color: #e74c3c;
    --pb-background: #ffebee;
    --pb-progress: 75%;
"></div>

Available CSS variables:

  • --pb-color - fill color (default: #3498db)
  • --pb-background - background color (default: #f0f0f0)
  • --pb-progress - progress value (default: 0%)

Themes: Blue Theme

First theme using CSS class:

<div class="progress-bar theme-blue" style="--pb-progress: 60%;"></div>

Features:

  • Custom blue color palette
  • Gradient effect on fill
  • Light blue background

Themes: Green Theme

Green theme with shadow effect:

<div class="progress-bar theme-green" style="--pb-progress: 80%;"></div>

Features:

  • Green color palette
  • Gradient with darker shade
  • Inner shadow for depth effect

Themes: Dark Theme

Dark theme with cyan accent for better visibility:

<div class="progress-bar theme-dark" style="--pb-progress: 70%;"></div>

Features:

  • Cyan color on dark gray background (high contrast)
  • Glow effect around fill
  • Inset shadow for depth
  • Border for definition

Size Modifiers

Different sizes for various use cases:

Small (10px):

Default (20px):

Large (30px):

Small with green theme:

<!-- Small size -->
<div class="progress-bar size-small" style="--pb-progress: 65%;"></div>

<!-- Large size -->  
<div class="progress-bar size-large" style="--pb-progress: 65%;"></div>

<!-- With theme -->
<div class="progress-bar theme-green size-small" style="--pb-progress: 80%;"></div>

Available size classes:

  • .size-small - 10px height
  • (default) - 20px height
  • .size-large - 30px height

Striped Pattern

Add striped pattern for visual effect:

Basic striped:

Striped with blue theme:

Small striped with green theme:

<!-- Basic striped -->
<div class="progress-bar striped" style="--pb-progress: 70%;"></div>

<!-- Striped with theme -->
<div class="progress-bar theme-blue striped" style="--pb-progress: 70%;"></div>

<!-- Multiple modifiers -->
<div class="progress-bar theme-green size-small striped" style="--pb-progress: 90%;"></div>

Features:

  • 45-degree striped pattern
  • Semi-transparent white stripes
  • Works with all themes
  • Can be combined with size modifiers
